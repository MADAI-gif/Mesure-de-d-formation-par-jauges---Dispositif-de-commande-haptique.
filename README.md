# Mesure-de-formation-par-jauges---Dispositif-de-commande-haptique.
Dans ce projet nous souhaitons mesurer l’effort appliqué au bout d’une lame en flexion par la mesure de la déformation d’une (ou des) jauge(s). Ensuite, l’effort mesuré servira de consigne pour le contrôle en position d’un servomoteur.
